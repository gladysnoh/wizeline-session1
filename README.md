# wizeline-session1
This repository was created in order to deliver my session 1 assessment for testcafe certification provided by wizeline
# How to use my test scripts
1. If you wanna run all tests in the project using the configuration file please run the following command:
`npm run test-from-config-file` 
   > It will run using firefox and chrome browsers and also will generate the report file in json format.
2. To run login cases use:
`npm run test-login-cases`
3. To run shopping cart cases use:
`npm run test-shopping-cases`
4. To run checkout cases use:
`npm run test-checkout-cases`
5. And finally to run overview cases use:
`npm run test-overview-cases`
